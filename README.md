# Hannah H. Chang - Academic CV

Academic CV written in R Markdown using the [stevetemplates](https://github.com/svmiller/stevetemplates) CV template.

## Usage

Open `HC_CV.Rmd` in RStudio and knit to PDF.

## Requirements

- R and RStudio
- LaTeX distribution (e.g., TinyTeX)
- `rmarkdown` package
